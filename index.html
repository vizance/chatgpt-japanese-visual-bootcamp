這場挑戰營在說什麼

這是一場專為「想開課、想做知識內容，但卡在視覺製作的講師 / 副業經營者 / 知識型工作者」設計的四週線上挑戰營。

由經營知識內容多年的朱騏教學，結合 ChatGPT 與日系設計的模組指令規格，透過四週循序漸進的工作流，讓你做出可以直接上架的日系課程封面、整套知識圖卡、銷售頁分區圖庫，以及一頁可以公開的課程銷售頁（ Landing Page）。

每週一場 60 分鐘直播，搭配 Discord 共學社群（連續四週），讓你回家真的把作品做出來，並公開在社群和課程頁面上。

四週後，你的電腦裡會多出這些

一條從文字 → 主視覺 → 圖卡 → 分區圖庫 → 銷售頁的完整工作流

W1

課程銷售頁主視覺工作流

從你「已經準備好的文字」出發，用 5 模組設計規格（文字內容 / 色彩計畫 / 字體特徵 / 佈局構圖 / 規格大小）拆解日系邏輯，再用一組封面 Prompt 模板生圖。沒有自己的文案也沒關係，朱騏會提供範例讓你練習。

成品： 1 張可上架的日系課程封面
W2

課程知識圖卡組工作流

用 8 段敘事弧線（ Hook / Gap / Vision / System / Transformation / Authority / Action / Final Call）規劃整套圖卡，再用 ChatGPT 批次生圖工作流一次產出，並學會修復文字重複、風格不一致等常見問題。

成品： 1 整套 10 張風格一致的日系知識圖卡
W3

銷售頁分區圖庫工作流

知識圖卡是「連續閱讀」，課程銷售頁是「分區塊閱讀」。把銷售頁拆成 Hero / 痛點 / 適合類型 / 收穫 / 贈品 / CTA 等標準區塊，用對應的 Prompt 模板，一次產出整套有區塊邏輯、風格統一的圖。

成品： 1 整套銷售頁分區圖庫
W4

課程銷售頁（ HTML）工作流

用「排版策略師」級別的 Prompt，把前三週的素材組成一份可貼進任何 CMS 的課程銷售頁 HTML。內建 6 種排版型態（ Z-Pattern / F-Pattern / Split-Screen / Asymmetric / Card Grid / Zig-Zag）與風格校準工作流，讓頁面色調跟你前三週的圖自動對齊。沒有 CMS 也行，會用 Netlify Drop 補充示範。

成品： 1 頁可以公開的課程銷售頁

這場挑戰營適合誰

  • 想開線上課程、課程內容都準備好了，卻卡在「課程封面怎麼做」的講師
  • 經營知識副業，每週都要產出視覺內容，但做出來的圖總是抓不到「日系專業」那種感覺、也看不出系列感的人
  • 用過 ChatGPT 生圖，知道它能生出漂亮的圖，但想把它變成「穩定產出整套視覺」工具的人

不是這三種類型也沒關係 —— 只要你手上有「文字」想轉成「視覺」（部落格文章、社群貼文、產品介紹、報告封面都可以），這套工作流就用得上。

關於講師 · 朱騏

我從 2019 年開始經營知識內容，累積超過 500 篇部落格文章、上萬名讀者，出版過《知識複利筆記術》和《復盤寫作術》兩本書。

我跟你一樣不是設計背景出身。會寫文章、會做課程、會講內容，但每次要做一張課程封面、一組社群圖卡就卡住 —— Canva 模板長得都很像、 Figma 學習曲線太陡、外包一張動輒幾千塊還不一定符合預期。

直到 ChatGPT Images 2.0 出來，我花了好幾個月把一張一張失敗的圖累積起來，才慢慢摸索出「日系設計 5 模組規格」與「敘事弧線 8 段結構」這兩套方法，現在它們是我自己課程封面、知識圖卡、社群貼文的穩定工作流。

這場挑戰營，就是把我走過的這幾個月彎路打包給你。
